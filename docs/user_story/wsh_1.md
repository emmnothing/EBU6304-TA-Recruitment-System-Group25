# User Stories
## TA Applicant
As a TA applicant
I want to create and manage my personal profile
So that I can submit applications for available teaching assistant positions.

## Module Organiser
As a module organiser
I want to publish and manage TA job postings
So that I can recruit suitable teaching assistants for my modules.

## Administrator
As an administrator
I want to view and balance the overall workload of all TAs
So that teaching assistant assignments are fair and reasonable.
