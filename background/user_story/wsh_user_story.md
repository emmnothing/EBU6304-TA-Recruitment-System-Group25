# User Stories

## TA Applicant
As a TA applicant  
I want to create and complete my personal profile  
So that I can apply for teaching assistant positions.

As a TA applicant  
I want to upload my CV document  
So that module organisers can review my application.

As a TA applicant  
I want to browse all available TA jobs  
So that I can find suitable positions to apply.

As a TA applicant  
I want to submit applications for open jobs  
So that I can be considered for recruitment.

As a TA applicant  
I want to check my application status  
So that I know the result of my submission.

---

## Module Organiser
As a module organiser  
I want to publish a new TA job posting  
So that I can recruit teaching assistants for my module.

As a module organiser  
I want to view all applications for my posted jobs  
So that I can screen and select qualified candidates.

As a module organiser  
I want to accept or reject applicants  
So that I can finalise the TA recruitment.

As a module organiser  
I want to update or close my job postings  
So that I can manage recruitment information correctly.

---

## Administrator
As an administrator  
I want to view TA workload statistics  
So that I can ensure workload is balanced and fair.

As an administrator  
I want to view all job postings and applications  
So that I can supervise the whole recruitment process.

As an administrator  
I want to manage user accounts and system data  
So that the recruitment system runs safely and normally.
