# User Stories

## TA Applicant

As a TA applicant  
I want to create a profile  
so that I can provide my personal and academic information for future job applications.

As a TA applicant,
I want to upload my CV,
so that module organisers can review my qualifications when I apply for a job.

As a TA applicant,
I want to browse available jobs and submit applications,
so that I can apply for suitable TA positions efficiently.

## Module Organiser

As a module organiser
I want to publish a TA job post
so that students can see the vacancy and apply for it.

As a module organiser
I want to review the list of applicants for my job post
so that I can compare candidates efficiently.

As a module organiser
I want to select suitable applicants for a TA position
so that I can recruit the best candidate for my module.

## Administrator

As an administrator
I want to view the overall workload of each TA
so that I can avoid assigning too much work to the same person.

As an administrator
I want to view all recruitment records
so that I can monitor the progress of the TA recruitment process.
