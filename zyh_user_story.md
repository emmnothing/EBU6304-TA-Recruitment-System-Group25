# User Stories

## TA Applicant

- As a TA candidate  
  I want to set up my personal profile  
  so that I can store my academic and contact details for applications.

- As a TA candidate  
  I want to submit my CV  
  so that lecturers can evaluate my qualifications.

- As a TA candidate  
  I want to explore open TA positions  
  so that I can identify roles that match my interests.

- As a TA candidate  
  I want to send applications for selected jobs  
  so that I can be considered for those positions.

- As a TA candidate  
  I want to view the progress of my applications  
  so that I can stay informed about decisions.


## Module Organiser

- As a module organiser  
  I want to create job listings  
  so that students are aware of available TA opportunities.

- As a module organiser  
  I want to access applicant information  
  so that I can assess their suitability.

- As a module organiser  
  I want to choose candidates for roles  
  so that I can assign the most appropriate TAs.


## Administrator

- As an administrator  
  I want to track the number of assignments per TA  
  so that workloads can be managed fairly.
